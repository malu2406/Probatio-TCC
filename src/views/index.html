Sobre o ENEM

Quando são abertas as inscrições?

As inscrições são abertas no final de maio e se estendem até o início de junho. Fique ligado no site oficial do GOV para saber os dias exatos da edição atual: site GOV

Qual o valor da taxa para a prova?

O valor da taxa para realização do ENEM é R$85. Mas você pode pedir isenção do valor da taxa se estiver dentro dos requisitos para tal.

Posso me inscrever sendo menor de idade?

Sim. Não existe idade mínima para se inscrever no ENEM. Mas se você for menor de 18 anos e não estiver concluindo o ensino médio, poderá participar como "treineiro".
